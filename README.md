## 🚀 快速開始

### 📋 環境需求

在開始之前，請確保你的電腦已安裝以下軟體：

| 軟體 | 版本 | 下載連結 | 用途 |
|------|------|----------|------|
| 🐍 Python | 3.7+ | [python.org](https://www.python.org/downloads/) | 執行主程式 |
| 📗 Node.js | 14+ | [nodejs.org](https://nodejs.org/) | 執行 Netlify CLI |
| 🌐 Netlify 帳號 | - | [app.netlify.com](https://app.netlify.com/) | 部署目標平台 |

```bash
# ╔════════════════════════════════════════════════════════════╗
```

## ❓ 常見問題

<details>
<summary><b>🔴 Q：出現「未偵測到 Netlify CLI」？</b></summary>

**A：** 請安裝 Netlify CLI：
```bash
npm install -g netlify-cli
```

安裝後重新啟動程式即可。

</details>

<details>
<summary><b>🔴 Q：登入失敗怎麼辦？</b></summary>

**A：** 請嘗試以下步驟：
1. 確認網路連線正常
2. 先到 [app.netlify.com](https://app.netlify.com/) 手動登入一次
3. 重新執行程式的登入功能

</details>

<details>
<summary><b>🔴 Q：部署失敗，顯示 "Function build failed"？</b></summary>

**A：** 常見原因：
1. `requirements.txt` 沒有放在 `functions` 目錄內
2. Python 套件名稱拼錯
3. 程式碼有語法錯誤

請檢查 `netlify/functions/requirements.txt` 是否存在且內容正確。

</details>

<details>
<summary><b>🔴 Q：環境變數沒有生效？</b></summary>

**A：** 設定環境變數後需要：
1. 到 Netlify 後台 → Site settings → Environment variables 設定
2. 重新部署網站

</details>

<details>
<summary><b>🔴 Q：GUI 版本無法啟動？</b></summary>

**A：** 請確認已安裝 Flask：
```bash
pip install flask
```

</details>

## 🛠️ 進階設定

### 自訂 netlify.toml

```toml
[build]
  functions = "netlify/functions"    # Functions 目錄
  publish = "dist"                   # 發布目錄
  command = "npm run build"          # 建置指令

[build.environment]
  PYTHON_VERSION = "3.10"            # Python 版本
  NODE_VERSION = "18"                # Node.js 版本

# API 路由重導向
[[redirects]]
  from = "/api/*"
  to = "/.netlify/functions/:splat"
  status = 200

# 自訂錯誤頁面
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200
```
